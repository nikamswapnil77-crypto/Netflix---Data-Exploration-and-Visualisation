# Netflix Data Exploration and Visualization

This project analyzes the Netflix dataset using Python (Pandas, NumPy, Seaborn, Matplotlib).

## Key Insights
- Movies make up ~70% of Netflix’s catalog, TV shows ~30%.
- USA dominates with 2000+ movies, India is second.
- Friday is the most common release day for both movies and TV shows.
- Top recurring actors include Anupam Kher, Shah Rukh Khan, and Akshay Kumar.
- Anime voice actors dominate TV show appearances, highlighting Netflix’s strong anime catalog.

## Files
- `Netflix_introduction.ipynb` → Jupyter Notebook with full analysis
- `Netflix - Data Exploration and Visualisation.pdf` → PDF report
